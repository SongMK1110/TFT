# 다음 개발 우선순위

## 현재 완성도

현재 프로젝트는 자동 전투 게임의 핵심 흐름이 연결된 상태다.

- 상점에서 유닛 구매, 벤치와 보드 배치, 유닛 판매
- 별 등급 업그레이드와 시너지 계산
- 적 편성 생성, 이동, 기본 공격, 스킬, 보호막, 전투 결과
- 라운드 보상, 경험치, 상점 확률, 저장과 불러오기
- 유닛 스프라이트, 스킬 이펙트, 전투 맵, 아이템 아이콘
- 아이템 선택, 클릭 장착, 드래그 장착, 보드 장착 아이콘

한 판을 실제로 진행할 수 있는 데모 수준은 넘겼다. 다음 목표는 전략 선택의 깊이와 반복 플레이의 재미를 높이는 것이다.

## 우선순위 1: 아이템 조합

기본 아이템 두 개를 조합해 완성 아이템을 만들면 선택의 밀도가 크게 올라간다.

- 조합표 데이터와 완성 아이템 정의 추가
- 유닛에 두 번째 기본 아이템을 장착할 때 조합 결과 확인
- 완성 아이템 전용 아이콘, 이름, 능력치, 특수 효과 표시
- 조합 전과 후의 능력치 차이를 장착 화면에서 확인

예시: 강철 검 + 신속의 활, 거인의 허리띠 + 수호자의 갑옷처럼 서로 다른 역할을 만드는 조합을 구성한다.

## 우선순위 2: 정보 가독성

게임 규칙을 외우지 않아도 판단할 수 있게 만든다.

- 유닛 카드와 상점 카드에 스킬 툴팁 추가
- 아이템 아이콘에 능력치와 효과 툴팁 추가
- 시너지 패널에 다음 단계까지 필요한 유닛 수 표시
- 아이템 드래그 중 장착 대상의 예상 능력치 변화 표시
- 전투 로그를 간단한 피해, 회복, 보호막 기록으로 정리

## 우선순위 3: 전투 연출과 조작

한 번의 전투가 더 읽기 쉽고 손맛 있게 보이도록 다듬는다.

- 전투 속도 조절과 일시 정지 버튼
- 스킬 시전 유닛과 대상의 짧은 강조 연출
- 치명타, 보호막 흡수, 상태 효과의 구분 강화
- 승리와 패배 결과 화면에 피해량, 회복량, 처치 수 요약
- 전투 중 유닛 선택 시 현재 장착 아이템과 능력치 표시

## 우선순위 4: 라운드와 난이도

반복 플레이에 목표와 변화를 만든다.

- 라운드별 적 조합과 아이템 보상 변주 확대
- 정예전과 보스전의 고유 패턴 추가
- 연승과 연패 보상을 화면에서 명확하게 안내
- 레벨 구간별 상점 확률과 추천 운영 흐름 표시
- 선택형 라운드 보상 또는 특수 이벤트 추가

## 우선순위 5: 품질과 테스트

현재 구현된 흐름이 기능 추가 후에도 안정적으로 유지되도록 한다.

- 유닛 재배치, 아이템 드롭, 전투 시작을 포함한 화면 흐름 테스트
- 기본 공격과 스킬이 같은 틱에 발생하는 전투 회귀 테스트
- 아이템 조합과 장착 제한 테스트
- 저장 파일 호환성 테스트 확대
- 전투 이펙트 중복 재생 여부를 확인하는 테스트 또는 디버그 도구

## 추천 진행 순서

1. 아이템 조합 데이터와 완성 아이템 6~10개 추가
2. 아이템과 유닛 툴팁으로 정보 가독성 개선
3. 전투 속도 조절과 전투 결과 요약 추가
4. 정예전, 보스전, 라운드 보상 변주 추가
5. 화면 흐름 테스트와 전투 회귀 테스트 확대
