# 아이템 조합표

## 기본 규칙

- 기본 아이템은 7종이다: 강철 검, 거인의 허리띠, 신속의 활, 신비한 지팡이, 수호자의 갑옷, 집중 부적, 마나 수정.
- 같은 아이템 두 개를 포함한 모든 조합을 지원한다.
- 조합 수는 `7 x 8 / 2`로 총 28개다.
- 유닛에 두 번째 기본 아이템을 장착할 때 해당 조합식이 있으면 재료 두 개를 완성 아이템 한 개로 교체한다.
- 완성 아이템도 유닛의 아이템 칸을 하나 사용한다.
- 완성 아이템은 다시 재료로 분해하거나 다른 아이템과 조합하지 않는다.

## 기본 아이템 역할

| 아이템 | 역할 | 기본 능력치 |
|---|---|---|
| 강철 검 | 물리 공격 | 공격력 +10 |
| 거인의 허리띠 | 생존 | 최대 체력 +150 |
| 신속의 활 | 기본 공격 | 공격 속도 +0.10 |
| 신비한 지팡이 | 주문 공격 | 스킬 위력 +15% |
| 수호자의 갑옷 | 방어 | 방어력 +12 |
| 집중 부적 | 빠른 시전 | 최대 마나 -10 |
| 마나 수정 | 초반 시전 | 시작 마나 +15 |

## 완성 아이템 28종

### 강철 검 조합

| 재료 | 완성 아이템 | 설명 | 구현 단계 |
|---|---|---|---|
| 강철 검 + 강철 검 | 처형자의 대검 | 공격력이 크게 증가하고, 처치 시 체력을 회복한다. | 특수 효과 |
| 강철 검 + 거인의 허리띠 | 거인 사냥꾼 | 공격력과 체력이 증가하며, 체력이 높은 적에게 더 강하다. | 특수 효과 |
| 강철 검 + 신속의 활 | 화염 연사검 | 공격력과 공격 속도를 함께 올리는 지속 공격 아이템. | 능력치 |
| 강철 검 + 신비한 지팡이 | 룬검 | 공격력과 스킬 위력을 높이는 하이브리드 아이템. | 능력치 |
| 강철 검 + 수호자의 갑옷 | 전장의 갑주 | 공격력과 방어력을 올리고, 체력이 낮아지면 보호막을 얻는다. | 특수 효과 |
| 강철 검 + 집중 부적 | 흡혈의 검 | 공격력이 증가하고, 준 피해 일부만큼 체력을 회복한다. | 특수 효과 |
| 강철 검 + 마나 수정 | 마력의 검 | 공격력과 시작 마나를 올려 빠른 첫 스킬을 노린다. | 능력치 |

## 구현 순서

### 1차: 능력치 조합 11종

화염 연사검, 룬검, 마력의 검, 워모그의 심장, 거인의 판금갑, 정령의 망토, 활력의 수정, 푸른 연사궁, 정찰대의 활, 대마도사의 모자, 푸른 정수를 먼저 구현한다.

기존 능력치 계산 시스템만 확장하면 되므로, 조합 시스템의 기본 흐름을 빠르게 검증할 수 있다.

### 2차: 전투 특수 효과 17종

보호막, 흡혈, 처치 회복, 부활, 추가 번개 피해, 지속 피해, 시전 강화, 마나 회복, 공격 속도 감소를 전투 엔진에 순서대로 추가한다.

특수 효과는 완성 아이템 한 종류씩 테스트와 이펙트를 함께 추가한다. 아이템 수는 많아도 각 효과의 책임 범위를 작게 유지한다.
