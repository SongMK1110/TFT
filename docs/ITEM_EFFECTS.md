# 아이템 특수 효과

## 얼어붙은 심장

- 조합식: 수호자의 갑옷 + 마나 수정
- 장착자 능력치: 방어력 +20, 시작 마나 +15
- 전투 시작 효과: 장착 유닛 기준 2칸 이내 적의 공격 속도를 20% 감소
- 중첩 규칙: 여러 오라 범위에 들어가도 가장 높은 감속 수치만 적용
- 시각 표시: 장착 유닛 주변에 옅은 푸른 냉기 오라 표시

## 구현 원칙

- 특수 효과는 전투 시작 시 아이템 능력치 적용 다음에 계산합니다.
- 조합 아이템은 재료 아이템 두 개를 대체해 장비 칸 하나만 차지합니다.
- 효과별 범위, 중첩, 지속 시간은 아이템 데이터와 전투 로직에서 명시적으로 관리합니다.
