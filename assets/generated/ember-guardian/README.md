# Ember Guardian Sprite Bundle

- `idle/sheet-transparent.png`: 2x2 transparent sheet, 4 frames, 128x128 per frame.
- `attack/sheet-transparent.png`: 2x3 transparent sheet, 6 frames, 128x128 per frame.
- `idle/frames/`: individual idle frame PNGs.
- `attack/frames/`: individual attack frame PNGs.
- `idle/ember_guardian_idle-strip-transparent.png`: 4-frame horizontal strip.
- `attack/ember_guardian_attack-strip-transparent.png`: 6-frame horizontal strip.
- `pipeline-meta.json`: source grid, shared scale, frame crop, and feet-anchor metadata.
